## 二. 创建版本库
1. git init - 初始化仓库 

    通过git init命令可以把当前目录变成Git可以管理的仓库。

1. git add - 添加文件到暂存区(stage）

1. git commit - 提交文件到仓库

1. git status - 查看git当前状态

1. git diff - 查看改变

1. git log - 查看日志

1. git reset - 回退、重置命令

    git reset --hard HEAD^ : 表示回退到上一次提交。
    git reset --hard \<commit id>: 回退到特定提交ID状态。

1. git reflog -- 查看命令记录
